2️⃣ What is the spread operator (...)?

  Answer:
  The JavaScript spread operator (...) expands iterable objects like array, string and objects into individual elements or properties in place where zero or more elements/arguments are expected.


3️⃣ What is the difference between map(), filter(), and forEach()?

  Answer:
  map() = this method transforms each element into a new array of the same length.
  filter() = this method selects a subset of elements into a new array based of a condition.
  forEach() = this method simply executes a function on each element without returning anything.
